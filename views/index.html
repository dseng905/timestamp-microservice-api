API Timestamp Microservice

This is an API that takes a data string compliant with ISO-6801 (e.g. 2020-1-1) and converts into JSON format.

The data string can be added to the URL to send a GET request. For example, [project_url]/api/timestamp/2020-2-2. Or, you add the data string into the input box below and submit it via a POST request

Once the data string is submitted, a page will be returned with the date in JSON format e.g. {"unix": 1479663089000 ,"utc": "Sun, 20 Nov 2016 17:31:29 GMT"}
